# 🚀 AI Code Analyzer (CodeSage)

An advanced AI-powered web application that analyzes code, detects bugs, explains logic, and suggests optimized solutions — all in a modern split-screen interface (VS Code + ChatGPT style).

---

## ✨ Features

### 🧠 AI Code Analysis
- Detects syntax & logical errors
- Provides detailed explanations
- Suggests optimized code
- Time & space complexity analysis

### 💻 Developer-Friendly UI
- Split-screen layout (Editor + AI panel)
- Chat-style AI responses
- Multi-language support (JS, Python, Java, etc.)

### ⚡ Smart Tools
- Code comparison (before vs after)
- Bug categorization
- Copy-to-clipboard
- Test data loader

### 🔄 Advanced Modes
- Full analysis
- Bugs only
- Optimization
- Explanation mode
